GW2 Elementalist Rotation Simulator and Build Optimizer

tool

This tool accurately simulates Elementalist DPS the same way the game calculates it. It doesn't use simplified formulas — it replays every combat event in order: weapon swings, skill casts, condition ticks, sigil procs, relic triggers, trait interactions, and attunement swaps.

  • Full build support — gear prefixes, runes, sigils, relics, food, utility, infusions, and all three traitlines with minor/major selections.
  • Accurate simulation — critical hits, condition stacking, boon tracking (Might, Fury, Alacrity, Quickness, etc.), and proc cooldowns are all modelled. Even handles Alacrity/Quickness loss mid-cast.
  • Expected values — no real RNG. Critical hit chance, weapon strength variance, and probabilistic procs use expected values, giving consistent, reproducible numbers.
  • Rotation builder — build your skill rotation manually and see the simulated DPS. Save and load entire builds (gear + traits + skills + rotation) as JSON.
  • Gear Optimizer — after building a rotation, run the optimizer to find the best gear combination (prefix mix, rune, sigil, relic, food, utility) for that exact rotation using coordinate descent.

Current limitations:

  • Skill list is intentionally incomplete — only DPS-relevant skills are included.
  • Summoned Elementals are not simulated.
  • No healing, barrier, or dodge simulation.

If you spot any bugs or have any suggestions, please report them or contact me on Discord at SnappyJoe#8710.
